# Palindrome Checker

This is a simple web application that allows users to check if a word entered is a palindrome or not. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward.

## Features

- Check if a word is a palindrome.
- Case-insensitive palindrome checking.
- Support for checking palindromes using the "Enter" key or clicking the check button.

## Usage

1. Enter a word in the input field.
2. Click the "Check" button or press the "Enter" key to check if the entered word is a palindrome.
3. The result will be displayed below the input field.
